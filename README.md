# Ubuntu Developer Setup

This repository bootstraps a fresh Ubuntu machine with all required development tools.

## Installed Software

- Homebrew
- Git
- NVM
- Node.js (LTS)
- npm
- pnpm
- Docker
- VS Code
- Cursor
- Postman

## Usage

```bash
chmod +x setup.sh
./setup.sh
```

After installation, logout/login once so Docker group permissions take effect.

Run verification:

```bash
./verify.sh
```
